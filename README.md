# Spring Boot CRUD API - 개발 중

이 프로젝트는 Spring Boot와 Spring Data JPA를 기반으로 한  
소프트웨어 엔지니어 관리용 CRUD API입니다.

현재는 기본적인 기능이 구현된 상태이며,  
향후 리팩토링 및 고도화를 진행할 예정입니다.

## ✅ 현재 기능

- 엔지니어 등록 (POST)
- 엔지니어 전체 조회 (GET)
- 엔지니어 ID 조회 (GET)
- 엔지니어 정보 수정 (PUT)
- 엔지니어 삭제 (DELETE)
